# slam_comparison

This code implements the metrics for vSLAM map (pointcloud) quality estimation as in our SibCon 2021 paper. If you use this code in your research, please cite this paper: TBA.

## Pre-requesites

2. PCL Library > 1.8
3. Octomap package
